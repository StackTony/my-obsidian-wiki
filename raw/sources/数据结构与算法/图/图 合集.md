# 算法图论必学全套
## 一、基础概念
1. 有向图、无向图
2. 有权图、无权图
3. 邻接矩阵、邻接表（**最常用存储**）
4. 顶点、边、度（入度/出度）
5. 连通图、连通分量、强连通分量
6. 环、自环、重边
7. 二分图、拓扑图、树图、森林
8. 路径、最短路径、最长路径

## 二、两大遍历（图论核心根基）
1. **BFS 广度优先搜索**
   - 层序遍历、最短无权路径、迷宫走法
2. **DFS 深度优先搜索**
   - 递归/栈实现、连通性、环判断、回溯

## 三、最短路算法（高频必考）
1. **Floyd 多源最短路**（暴力三层循环）
2. **Dijkstra 单源最短路**（无负权，堆优化）
3. **Bellman-Ford 负权可跑**
4. **SPFA 队列优化BF**（日常最常用）

## 四、拓扑排序（工程、任务依赖必学）
- 入度表 + BFS拓扑
- 判环、任务编排、编译顺序、依赖解析

## 五、最小生成树 MST
1. **Prim 算法**
2. **Kruskal 并查集实现**
- 场景：布线、组网、成本最小连通

## 六、并查集 Union-Find（图论神器）
- 合并集合、查找根节点
- 连通性判断、朋友圈、岛屿、Kruskal必备

## 七、连通性进阶
1. 无向图连通分量
2. **Tarjan 强连通分量 SCC**（有向图缩点）
3. 割点、割边（桥）

## 八、二分图专题
1. 二分图判定
2. **匈牙利算法 二分图最大匹配**
3. 最大流最小点覆盖

## 九、网络流（进阶中高阶）
1. 最大流 Dinic
2. 最小割
3. 费用流

## 十、经典模型（刷题直接套）
- 岛屿数量、省份数量
- 迷宫最短路径
- 课程表（拓扑）
- 朋友圈（并查集）
- 物流最短路
- 行程规划、路径规划
- 二分图匹配排班

## 十一、代码必备写法
- 邻接表建图
- BFS模板
- DFS递归+迭代模板
- 堆优化Dijkstra
- SPFA
- 拓扑排序模板
- 并查集模板
- Tarjan模板

## 十二、学习顺序（最简路线）
**图基础 → 邻接表 → BFS/DFS → 并查集 → 最短路 → 拓扑 → 生成树 → 强连通 → 二分图 → 网络流**
